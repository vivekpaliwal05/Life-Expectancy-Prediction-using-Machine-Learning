# Life-Expectancy-Prediction-using-Machine-Learning
Machine Learning project that predicts life expectancy and health risk based on factors such as age, BMI, blood pressure, cholesterol, diabetes, smoking, and alcohol consumption.

## Features
- Life Expectancy Prediction
- Health Risk Analysis
- Data Visualization
- Power BI Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Power BI
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Prediction
6. Dashboard Visualization

## Files
- final_pro.ipynb : Machine Learning Model
- Workbook1.xlsx : Dataset
- PATIENT.pbix : Power BI Dashboard
